# Renaissance-5

## This is a python game in <a href="https://t.me/Renaissance_beta_bot">telegram bot</a>

### Description
Telegram bot game written on python, which you can run by yourself with your game data

### Instructions for develop
- Fork this repository
- Clone your forked repository
- Add your scripts
- Commit and push
- Create a pull request
- Star this repository
- Wait for pull request to merge

### Instructions for use
- Clone this repository
- Run 'python3 init.py' in 'utilities/'
- If you have access to original game data (you need special token) run 'pyhon3 download_game_data.py'
- Write game data to right directories with right syntax
